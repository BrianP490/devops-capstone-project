the command to show color and run coverage with nosetests is:
    nosetests -vv --with-spec --spec-color --with-coverage --cover-erase --cover-package=service

compare branches
    git diff main..<feature-branch>

delete a feature branch from local
    git branch -d <feature-branch>

delete a feature branch from remote
    git push origin --delete <feature-branch>
